# 해결기록 사용 가이드

이 폴더는 아직 완전히 해결되지 않은 문제, 해결 과정, 그리고 나중에 다시 확인해야 할 내용을 기록하는 곳이다.

## 이 문서의 목적

- 해결되지 않은 문제를 잊지 않게 한다
- 진행 중인 이슈를 주제별로 분리해 둔다
- 나중에 해결 방법과 검증 결과를 함께 남긴다

## 작성 원칙

- 한 문서는 하나의 문제만 다룬다
- 문제, 원인 추정, 진행 방향을 먼저 적는다
- 해결되면 해결 방법과 검증 결과를 보강한다
- 코드가 바뀌면 관련 파일과 이유를 같이 적는다

## 추천 항목

- 문제 상황
- 재현 방법
- 현재까지 확인한 내용
- 시도한 해결책
- 다음 행동
- 검증 결과
