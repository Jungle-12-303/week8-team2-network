## 용도

- 다른 브랜치와 현재 브랜치를 비교할 때
- 리뷰 의견을 반영한 결과를 기록할 때
- 앞으로 발견할 문제와 해결 과정을 남길 때

## 해결기록 작성 기준

- 아직 해결되지 않은 문제도 기록할 수 있다.
- 주제별 문서는 하나의 문제만 다룬다.
- 해결 전에는 문제, 학습 메모, 진행 방향만 적어도 된다.
- 해결 후에는 해결 방법, 검증 결과, 남은 과제를 보강한다.
