# 생성된 플랜 문서 안내

이 폴더는 개발 중 새로 만들어진 플랜 문서를 모아 두는 곳입니다.

## 목적

- 정규 플랜 문서와 분리한다
- 작업 중 새로 생긴 계획 문서를 모아 둔다
- 테스트, 마이그레이션, 실행 가이드처럼 변동이 잦은 문서를 따로 관리한다

## 기준

- `docs/plan/*.md`는 프로젝트의 정규 계획 문서다
- `docs/plan/generated/*.md`는 개발 과정에서 새로 만든 계획 문서다
- `docs/test/demo/`는 설명용 문서만 둔다
