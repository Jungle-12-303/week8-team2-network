# 동시성 테스트 시나리오 정리함

## 목적

동시 요청이 들어왔을 때 서버가 어떻게 동작하는지 검증 항목을 만든다.

## 범위

- 다중 `INSERT` 확인함
- 다중 `SELECT` 확인함
- 혼합 요청 확인함
- queue full 확인함
- 서버 종료 중 요청 확인함

## 완료 기준

- 경쟁 조건을 드러내는 시나리오가 있음
- 재현 방법이 문서화됨
- 실패 시 기대 결과가 정의됨

## 테스트/검증

- 병렬 curl로 확인함
- 결과 누락 여부를 확인함
