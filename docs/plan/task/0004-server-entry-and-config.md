# 서버 진입점과 설정 정리함

## 목적

서버가 어떻게 시작되고 어떤 설정을 읽는지 정리한다.

## 범위

- 서버 main 진입점 확인함
- 포트 설정 방식 정리함
- 환경값 또는 인자 처리 방식 정리함
- 종료 시그널 처리 기준 정리함

## 완료 기준

- 서버 시작 흐름이 한눈에 보임
- 설정 위치가 명확함
- 종료 조건이 정의됨

## 테스트/검증

- 서버가 기동되는지 확인함
- 포트 충돌 상황을 확인함
